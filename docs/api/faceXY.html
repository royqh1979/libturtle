主页-＞库函数目录-＞faceXY

功能：
设置海龟转动朝向点(x,y)（有动画）
注意：0度表示海龟朝向屏幕正右方, 90度表示海龟朝向屏幕正上方, -90度表示海龟朝向屏幕正下方.

声明：

void faceXY(double x, double y);


参数：
x
要朝向的点的横坐标
y
要朝向的点的纵坐标


返回值：
（无）

示例：

#include <turtle.h>

int main() {
    initWorld(800,600,1);
    setSpeed(50);

    setXY(0,100);
    faceXY(-200,100);
    fd(100);
    home();
    pause();
    hide();
    pause();
    return 0;
}
