主页-＞库函数目录-＞clearScreen

功能：
清除屏幕，并让海龟返回初始位置（屏幕正中央，海龟朝向屏幕正上方）。

声明：

void cs();
void clear();
void clearScreen();



参数：
（无）

返回值：
（无）

示例：

#include <turtle.h>

int main() {
    initWorld(800,600,1);
    setSpeed(50);

    rightTurn(90);
    fd(100);
    rt(90);
    fd(200);
    home();
    wait();
    cs();
    wait();
    return 0;
}
