主页-＞库函数目录-＞setImmediate

功能：
设置是否取消海龟移动动画（立即显示最终图形）

声明：

void setImmediate(int isImmediate);


参数：
isImmediate
是否立即显示最终图形，1为立即显示，0为不立即显示。缺省为不立即显示。

返回值：
（无）

示例：

#include <turtle.h>

int main() {
    initWorld(800,600,1);
    setImmediate(1);

    fd(100);
    lt(90);
    fd(200);
    home();
    wait();
    hide();
    wait();
    return 0;
}
