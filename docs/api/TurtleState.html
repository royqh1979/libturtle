主页-＞库函数目录-＞TurtleState结构说明

作用：
用于表示海龟的位置和朝向。
见坐标系和海龟朝向说明

声明：


typedef struct {
    double x;
    double y;
    double angle;
} TurtleState;



成员变量：
x
海龟位置的x坐标值
y
海龟位置的y坐标值
angle
海龟的朝向

    示例：

#include <turtle.h>

int main() {
    initWorld(800,600,1);
    setSpeed(50);

    setAngle(45);
    fd(100);
    lt(90);
    fd(200);
    home();
    pause();
    hide();
    pause();
    return 0;
}
