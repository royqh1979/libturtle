主页-＞库函数目录-＞turnTo

功能：
让海龟转到指定朝向（有动画）
注意：0度表示海龟朝向屏幕正右方, 90度表示海龟朝向屏幕正上方, -90度表示海龟朝向屏幕正下方.

声明：

void turnTo(double angle);


参数：
angle
海龟的朝向

返回值：
（无）

示例：

#include <turtle.h>

int main() {
    initWorld(800,600,1);
    setSpeed(50);

    turnTo(45);
    fd(100);
    lt(90);
    fd(200);
    home();
    wait();
    hide();
    wait();
    return 0;
}
