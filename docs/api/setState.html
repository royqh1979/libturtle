主页-＞库函数目录-＞setState

功能：
设置海龟位置和朝向，原位置和新位置之间不会留下痕迹。主要和getState函数配合使用，保存和恢复海龟位置朝向。
如果需要留下移动痕迹，请使用setXY、gotoXY和setAngle
见坐标系和海龟朝向说明以及TurtleState结构说明

声明：

void setState(TurtleState state);


参数：
state
海龟的新位置和朝向

返回值：
（无）

示例：

#include <turtle.h>

int main() {
    initWorld(800,600,1);
    setSpeed(50);

    setAngle(45);
    fd(100);
    lt(90);
    fd(200);
    home();
    wait();
    hide();
    wait();
    return 0;
}
