主页-＞库函数目录-＞setXY

功能：
直接设置海龟位置为点(x,y)（没有动画）,不改变海龟朝向
    注意：海龟初始位置为（0,0），横坐标正向为屏幕向右方向，纵坐标正向为屏幕向上方向

声明：

void setXY(double x, double y);


参数：
x
要移动到的点的横坐标
y
要移动到的点的纵坐标


返回值：
（无）

示例：

#include <turtle.h>

int main() {
    initWorld(800,600,1);
    setSpeed(50);

    setXY(0,100);
    setXY(-200,100);
    home();
    pause();
    hide();
    pause();
    return 0;
}
