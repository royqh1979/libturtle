主页-＞库函数目录-＞initWorld

功能：
这个函数用于初始化海龟绘图系统。在每一个海龟绘图程序中，都必须先运行该函数，才能正确执行其他的作图指令。

声明：

void initWorld(int width, int height, double scale = 1.0);


参数：
width
绘图区的宽度（实际显示窗口的大小为width*scale)
height
绘图区的高度（实际显示的窗口大小为height*scale）
scale
显示缩放比例，缺省为1

返回值：
（无）

示例：

#include <turtle.h>

int main() {
    initWorld(800,600,1);
    setPenColor(RED);
    setSpeed(100);

    fd(100);
    lt(90);
    fd(200);
    home();
    pause();
    hide();
    pause();
    return 0;
}
